🔒 Politique de confidentialité

📊 Données collectées
• Code UAI de l'établissement (identifiant public).
• Contenu scanné (codes-barres, textes saisis).
• Mode d'utilisation (solo ou collaboratif).
• Dates de première et dernière utilisation.

🎯 Finalité
• Données de récolement : permettre le récolement collaboratif au sein de l'établissement.
• Statistiques : connaître le nombre d'établissements utilisant l'application.

📍 Stockage
• Données de récolement : hébergées sur Google Cloud Firestore (Europe) en mode collaboratif.
• Données statistiques : hébergées sur Google Cloud Firestore (Europe).

⏱️ Durée de conservation
• Données de récolement : conservées tant que l'établissement utilise l'application.
• Données statistiques : conservées à des fins d'analyse.

🗑️ Vos droits
Vous pouvez supprimer vos données à tout moment via le bouton "Réinitialiser" dans l'application.
